# For, While, Do-while Loop


<b>For loop</b>

```
Console.Write("For: ");
int i;
for(i=0; i<10; i++)
{
Console.Write(i);
}
```

<b>While loop</b>

```
Console.WriteLine(" ");
Console.Write("while: ");
i = 0;
while (i < 10)
{
Console.Write(i);
i += 1;
}
```

<b>Do while loop</b>

```
Console.WriteLine(" ");
Console.Write("Do while: ");
i = 0;
do
{
Console.Write(i);
i += 1;
}while(i < 10);
```
